Introduccion

DL-JS es una interfaz gráfica del programa youtube-dl, la cual incorpora una serie de preconfiguraciones diseñadas para el usuario final. Este programa busca facilitar la descarga de multimedia en diversos sitios web sin la intromisión de terceros.

La aplicación está en estado beta (Cubre todas las características objetivo mas no ha sido testeada), por lo que se inicia con una consola

El programa está Basado en español (Its partialment traslated to English) y se inicia el inglés si la maquina tiene el idioma ingles por defecto (Parcialmente traducido). Futuras traducciones serán añadidas

Por el momento La aplicación solo está disponible para Windows, más tiene el potencial para Linux y Mac (Sin testeo a fondo), Además de un port para Android para futuro desarrollo.

Fundamentos Básicos

Al ejecutar la interfaz se muestra un bloque para colocar el texto y tres opciones visible. Para acceder a ellas necesario colocar un URL valido y seleccionar el modo de descarga (Audio - Video - Subtítulos).

Los URL no deben contener espacios y deben ser 1 URL por vez.

www.ejemplo.com

al terminar se muestra el mensaje Completado si todo transcurrió con normalidad.

sí al terminar muestra error puede que no se haya terminado de descargar el video completamente, no se haya podido convertir, haga falta un video en la lista de reproducción etc. Al terminar se muestra debajo el comando usado en caso de Error y si se está en modo Debug.
    Modos:
  • Modo Audio

    Este descargara el audio (si está disponible) del URL seleccionado en la mejor calidad y si además están disponibles se le añaden los metadatos y caratula del audio (Cover). Y convierte en MP3 el audio

    De no estar FFMpeg disponible no se aplicarán todas las características (Preinstalado por defecto)
  • Modo Video

    Este descargara el video en la mejor calidad posible

    En el caso de Crunchyroll se le añaden los subtítulos embebidos, no superpuestos. Si el audio y video de mejor calidad están disponibles por separado se descargarán y unirán al finalizar la descarga. Al forzar Metadatos en video se convertirá en formato MKV lo cual puede tardar un tiempo durante la conversión.
  • Subtitulos

    Este descargara los subtítulos en Idioma español de forma predeterminada (Español Latino si no España), para cambiar esto Accede a configuración.

    • Si la mejor calidad no está disponible se procederá a la segunda mejor y así sucesivamente

Configuración

En la esquina superior derecha se encuentra un icono con forma de engranaje, el cual muestra las configuraciones, además del cuadro de inicio de sesión y acceso a las actualizaciones (Aun en desarrollo)

    Configuraciones
  • Cuadro de Logeo

    Permite colocar los datos para iniciar sesión en el caso de que se requiera un usuario para descargar algún video.

    Todo inicio de sesión se hace al empezar la descarga y se cierra al culminar esta.

    La configuración se borra al reinicio

    Ningún dato es almacenado o enviado a terceros en ningún momento. Puede que el Login requiera resolver algún Recapta, en dicho caso de debe usar un navegador
  • ComoBoxs de Calidad

    La descarga automáticamente elegirá la mejor combinación de calidad posible, pero en dicho caso se puede elegir un límite de calidad para el video (Resolución y FPS) y audio (AudioRate). Además, se puede seleccionar el idioma de los subtítulos.

    Si dicha configuración no está disponible se pasa a la mejor disponible (o a la más cercana)
  • Añadir Metadatos

    Predeterminado en modo audio, añade la información disponible sobre el medio. Nombre, autor, Álbum, año, etc.

    Solo si FFMpeg está disponible (Pre instalado)
  • Añadir Caratula

    Añade el Cover correspondiente en el medio seleccionado. Como la caratula de un álbum

  • Restringción de Nombre

    Evita que se creen archivos con nombres no compatibles en Windows (Activado por defecto)

    No tiene efecto en Linux o Android
  • Nombre Automatico

    El nombre se Obtiene mediante el título del medio. De no estar activado se pide el nombre del archivo No requiere el nombre de la extensión.

    Soporta Plantillas de salida como %(playlist_index)s - %(title)s.%(ext)s (NumeroDeLista - Nombre.extencion Eg. 11 - thank u, next.mp3)
  • Forzar Metadatos

    Añade los Metadatos y caratula a videos

    Suele convertir los videos a formato MKV lo que requiere de más tiempo
  • Forzar Subtítulos

    Añade los subtítulos al video (Automático para Crunchyroll)

  • Modo Album

    Descarga Listas de Reproducción, Canales, (Listas de Listas de reproducción), recursivamente, y las ordena según el nombre del álbum (o lista de reproducción), numero de lista y titulo. en carpetas separadas

    Si se selecciona un canal o usuario descargara todo su contenido. Úsese con cuidado
    Para Dumpear (Descargar todo) un canal de música se recomienda usar el listado de Álbumes & Canciones este se obtiene desde el canal de youtube (Solo en canales que están también en Youtube Music)
    Suele tener la siguiente forma:

    https://www.youtube.com/c/muse/playlists?view=71
  • Resetear

    Resetea a la configuracion de fábrica

Comandos Heredados Útiles

Son una serie de comandos que se pueden añadir antes del URL que pueden ser útiles en ciertos casos. Estos comandos son heredados del YoutubeDL y FFmpeg Solo Desarrolladores

  • No descarga ningún medio o quiero un formato especifico

    Para conocer la lista de formatos disponible antecede -F antes del URL

    -F www.ejemplo.com

    Este retorna a una lista de formatos en la consola cada uno con una ID y formato. para seleccionarlo usa -f ID. Por Ejemplo

    -f 140 www.ejemplo.com
  • Para descargar los videos específicos de una lista de reproducción

    Para empezar desde un numero especifico usa --playlist-start Nro

    --playlist-start 3 www.ejemplo.com

    Para descargar hasta cierto número usa --playlist-end Nro

    --playlist-end 12 www.ejemplo.com

    Para descargar algunos videos específicos usa --playlist-items Nro. separando los números con (,) y marcando rango con (-)

    --playlist-items 1,3,13,15 www.ejemplo.com --playlist-items 1-3,13,15-17 www.ejemplo.com

    Para descargar en orden inverso usa --playlist-reverse

    Para descargar en orden aleatorio usa --playlist-random

  • No se añadieron los subtitulos Reportar si es posible

    en algunas páginas el ID del idioma del subtitulo no es compatible por lo que se debe ingresar manualmente.

    Primero se deben obtener la lista de subtítulos con --list-subs

    --list-subs www.ejemplo.com

    En la consola se obtendrá una lista de los subtítulos por ID e Idioma-Tipo


    Luego se debe usar --sub-lang Lenguaje con ID seleccionado

    --sub-lang "ES_es-419" www.ejemplo.com

    También se puede usar directamente --all-subs para descargar todos los subtítulos disponibles

    --all-subs www.ejemplo.com

    En el caso de los subtítulos autogenerados de youtube se usa --write-auto-sub

    --write-auto-sub www.youtube.com/watch?v=

    Los subtítulos embebidos al video solo se activan automáticamente en crunchyroll, para añadirlos automáticamente usa la opción Forzar Subtítulos en la configuración.

  • Esto no es MP4

    el video se descarga en el formato disponible por la página en el caso de que se descargue en otro formato y quieras convertir el video automáticamente usa --recode-video Formato

    --recode-video MP4 www.ejemplo.com Soporta mp4 flv ogg webm mkv avi
